Tic Tac Toe

El objetivo de este proyecto es que el usuario pueda disfrutar del conocido juego de tic tac toe de manera virtual, jugando vs la computadora.

Reglas detalladas:

1. Tablero y piezas:
   El juego se juega en un tablero de 3x3 casillas. Cada jugador elige su marca (X o O).
2. Turnos:
   Los jugadores se turnan para colocar sus marcas en una casilla vacía.
3. Objetivo:
   El objetivo es ser el primero en colocar tres de sus marcas en línea (horizontalmente, verticalmente o diagonalmente).
4. Fin del juego:
   El juego termina cuando un jugador logra tres en raya o cuando todas las casillas están llenas y no hay ganador.
5. Empate:
   Si todas las casillas están llenas y ningún jugador ha logrado tres en raya, el juego termina en empate.
